ATM

Input Card

  • DISPLAY, "Please enter your card?"
  • INPUT your card
  • SET cardCheck equal to CALL checkCard to pass it
  • IF cardCheck returns True
  • Goes to PIN
  • IF cardCheck returns False
  • Goes to PIN and applies a fee

Pin

  • Display, "Please enter your pin?"
  • INPUT your pin
  • SET pinCheck equal to CALL checkPin to pass it
  • IF pinCheck returns True
  • Goes to Dashboard
  • IF pinCheck returns False
  • DISPLAY, "Invalid pin please try again."
